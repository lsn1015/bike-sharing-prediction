#### \# Bike Demand Prediction

#### 

This project predicts daily bike rental demand using Python and machine learning. It's a great choice for new learners who want to practice machine learning basics.  

The baseline model uses \*\*Linear Regression\*\*. The project includes data preprocessing, model training, evaluation, and visualization.



* Dataset source: \[Kaggle - Bike Sharing Dataset](https://www.kaggle.com/c/bike-sharing-demand)
